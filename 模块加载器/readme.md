1. 全局作用域会变得很混乱.

### 依赖分析的实现

假设存在 A --> B --> C --|
              |         |-----> E
              |---> D --|
              
E 不存在依赖的关系, 只被其他模块依赖.

对于其他模块, 该模块运行的前提是它所依赖的模块已经被执行完.

触发顺序是 E --> C或者D --> B --> A 

通过对代码进行分析,找到每一个模块的依赖之后, 就需要对所有的模块做依赖分析. 
找到整个依赖结构中没有依赖的模块.
